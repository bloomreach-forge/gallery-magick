Gallery Magick Image CMS Plugins

Introduction

Gallery Image Processing CMS Plugins provides plugins for Hippo CMS Authoring application, using Gallery Image Processing Core Library.

This library provides the following plugin components:

  • MagickCommandGalleryProcessorPlugin : GalleryProcess service plugin which reads configurations, instantiates MagickCommandGalleryProcessor and register it as a GalleryProcessor service component for Hippo CMS Authoring application.
  • MagickCommandGalleryProcessor : GalleryProcessor service component implementation.
